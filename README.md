# FastAPI CRUD Test Project

This is a simple FastAPI project created for testing and practicing CRUD (Create, Read, Update, Delete) operations.

## 🚀 Features

- ✅ Create item
- 📖 Read one or all items
- ✏️ Update item
- 🗑️ Delete item
- 🧪 Easy to test and expand

## 🧱 Stack

- Python 3.10+
- FastAPI
- Uvicorn
- Pydantic
